# [Realm](https://realm.io/cn/)

Realm 是一种开源**对象数据库管理系统**，最初**用于移动操作系统**，但也可用于 Xamarin，React Native 等平台（包括桌面应用程序），并根据 Apache 许可证获得许可。

ps：目标是取代作为 SQLite 或 Core Data 的替代方案，

应用场景

- 离线应用
